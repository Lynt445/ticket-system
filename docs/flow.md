## Flow

user enters details in the frontend
data sent to google sheet
payment intiated (mpesa)
data sent to pabbly/n8n/zapier(automation tool)
data sent to qr.io(QR generation tool)
qr sent to the user

### Tools Used

1. Database

   - google sheet(holds the user details)
   - google drive(holds the qr codes and ticket with qr codes)
   - google doc(template doc ticket holding)

2. QR generation tool

   - qr.io

3. Automation tool

   - pabbly/n8n/zapier

4. next js

   - frontend(UI)

5. Express js
   server
